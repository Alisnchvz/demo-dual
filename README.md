# Demo.Dual
Seguimiento del proyecto de demostración dual
¿En qué consiste el proyecto?

Mi proyecto facilita el trabajo de mi padre, el cual ejerce la profesion de carpinteria y con mi proyecto busco facilitar dos cosas:
1. Cotizacion de productos
2. Venta de los producto
   
Mi proyecto se basa en ventas en linea, creando un sitio web donde los productos de mi padre son mostrados y las personas interesadas puedan comprar o hacer un encargo en un apartado del sitio web.
